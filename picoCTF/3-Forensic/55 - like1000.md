### like1000

This [.tar file](https://challenge-files.picoctf.net/c_fickle_tempest/dc6349af33fec5413fe968b12506dde81953fdab2214acb3eaaf0b96cfff5dc4/1000.tar) got tarred a lot.
### Solución 
Se uso un script para descomprimir los archivos tar de 1000 a 1
```
┌──(kali㉿kali)-[~/problemasPICO/like1000]
└─$ for i in {1000..1}; do tar -xvf $i.tar; done                                                                      
999.tar
filler.txt
998.tar
filler.txt
997.tar
....
```
